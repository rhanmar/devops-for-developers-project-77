# Команды
* `make tf-init` - инициализация зависимостей Terraform
* `make tf-apply` - создание инфраструктуры
* `make tf-validate` - валидация скриптов
* `make tf-destroy` - удаление инфраструктуры
* `make ansible-setup` - установка зависимостей
* `make ansible-deploy` - деплой приложения
* `make ansible-datadog` - установка мониторинга
